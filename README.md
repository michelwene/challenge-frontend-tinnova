A solução é uma listagem de usuários com formulário de cadastro/edição.

## Instalação

```
git clone https://github.com/michelwene/tinnova-teste.git
ou
git clone git@github.com:michelwene/tinnova-teste.git
cd tinnova-teste
yarn build
yarn start
```
